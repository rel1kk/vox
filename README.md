# VOX — Социальная сеть

## Запуск (Windows)

1. Открой папку `vox` в проводнике
2. В адресной строке папки напиши `cmd` → Enter
3. Выполни:

```
npm install
npm start
```

4. Открой браузер: http://localhost:3000

## Деплой на Railway

1. Загрузи папку на GitHub
2. Зайди на railway.app → New Project → GitHub repo
3. Выбери репо → Generate Domain → готово!

## Данные
Все данные хранятся в файле `vox-data.json` — он создаётся автоматически.
